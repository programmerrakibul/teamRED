# teamRED

Welcome to the **teamRED** portfolio website! This project showcases our team's
collaborative development efforts, combining creativity and technical skills to
build a dynamic web application.

## 📝 Overview

**teamRED** is a project developed to demonstrate our web development
capabilities, including front-end design, functionality, and responsiveness. The
website highlights our team's work, projects, and other relevant information.

## 💻 Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Swiper.js

## 🚀 Features

- Modern, responsive design
- User-friendly navigation
- Dynamic content display
- Contact form
